React Hooks

Posted by Scott Sandman on 2020-04-28

Last week at JRS Code School, we began building web applications with React. In a lot of ways, this was a refresher for me, having taken an online mobile app development course which utilized React Native. But, as mostly is the case, there are multiple solutions to every problem. In the aforementioned class and other classes I have taken, the approach has been to use Object Oriented Programming (OOP) utilizing the Class structure. However, for the past several weeks, our studies at JRS have been focused on using a functional approach as our solutions. React hooks fit nicely into that approach.

First, let's take a look at a class method which uses props and state:

As you can see, the class method uses a constructor to handle the props and set the initial state for firstName and lastName which are then called by this.state.firstName and this.state.lastName.

Now let's take a look at how we can do this using hooks:

To start, I imported and deconstructed the hook useState from React. Then I simply set up a const variable in the form of an array which takes the variable name and a method to update that variable and set that equal to the hook useState where I was able to set the initial value for firstName. LastName is set up exactly the same way. Now, instead of having to use this.setState to update firstName or lastName, I simply just need to implement the method:

setFirstName("Hank")

That's all for this iteration. Next up I will dive into the hook useEffect. After that, we'll check out redux. Yet another solution to handling state. Until then, I'll leave you with the following quote by John Wooden, "It's what you learn after you know it all that counts."
